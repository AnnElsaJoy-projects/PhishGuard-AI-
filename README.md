# 📖 About The Project

PhishGuard AI is a cybersecurity application that uses **Artificial Intelligence and Machine Learning** to detect phishing emails.

The system analyzes email content, identifies suspicious patterns, and predicts whether the email is:

- 🔴 **Phishing / Malicious**
- 🟢 **Safe / Legitimate**

The application provides users with a modern security dashboard containing risk analysis, confidence scores, threat indicators, and scan analytics.
